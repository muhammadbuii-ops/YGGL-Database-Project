# YGGL-Database-Project
从零开始，系统实践了数据库环境搭建、表结构设计、数据增删改查（CRUD）、复杂业务逻辑查询及数据视图封装，构建了一个可用于模拟企业人事与薪酬分析的数据基础，全面夯实了作为数据分析师所必需的数据底层操作与理解能力。
# 企业人力资源数据分析库全流程构建与实战 | YGGL Database Project

## 项目概述
独立完成从零搭建企业人力资源数据库到复杂数据分析的全流程项目。  
主要包括：数据库环境搭建、数据建模、CRUD操作、复杂查询及视图封装。

## 技术栈
- MySQL + Navicat
- SQL (DDL, DML, DQL)
- 数据建模、查询优化

## 核心模块
1. **数据建模与建库**
   - 员工表 (Employees)
   - 部门表 (Departments)
   - 薪资表 (Salary)
   - 完整定义主键、外键约束与数据类型

2. **数据维护与质量保障**
   - 插入、更新、删除模拟数据
   - 实践外键约束与完整性规则

3. **核心查询与分析**
   - 50+ SQL 查询任务，包括：
     - 单表筛选
     - 多表连接与聚合
     - 分组统计与子查询
   - 常用查询封装为视图，提高分析效率

## 亮点
- 全流程实践，端到端数据操作能力
- 扎实 SQL 技能：熟练使用 JOIN、GROUP BY、CASE WHEN、子查询
- 数据工程思维：数据质量、复用、抽象层面理解
- 业务映射能力：薪酬分析、员工统计、部门查询

## 文件说明
- `sql/` : 数据库脚本
- `docs/` : ER 图、表结构图、数据示例
- `outputs/` : 查询结果示例
- `INTERVIEW.md` : 面试项目解说
