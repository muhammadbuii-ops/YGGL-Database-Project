# 数据库综合项目面试解说｜YGGL 人力资源系统

## 一、项目背景
该项目模拟企业人力资源管理系统，目标是从零构建一个可用于数据分析的结构化数据库环境，用于员工、部门与薪资的统一管理和分析。

---

## 二、我在项目中的角色
**独立完成整个数据库项目的设计、实现与分析**

---

## 三、项目实施过程

### 1️⃣ 数据库设计与建模
- 根据业务需求设计员工表、部门表和薪资表
- 明确主键与外键关系，保证表之间的业务关联
- 设置合理的数据类型与约束条件

### 2️⃣ 数据维护与完整性控制
- 手动插入模拟业务数据
- 使用 UPDATE 和 DELETE 实现条件数据维护
- 通过外键约束，确保数据操作符合业务逻辑

### 3️⃣ 核心 SQL 查询与分析
- 完成 50+ 条 SQL 查询任务
- 包括：
  - 单表与多表 JOIN 查询
  - 聚合分析（人数、平均薪资等）
  - 子查询解决复杂业务问题
- 示例业务问题：
  - 统计各部门员工数量
  - 查找薪资高于某部门所有员工的人员
  - 分析不同学历的员工分布情况

### 4️⃣ 数据视图封装
- 将常用复杂查询封装为 VIEW
- 简化后续分析流程
- 提高 SQL 复用性和可维护性
